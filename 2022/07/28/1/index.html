寻找图床的记录

话说在找图床的道路上，我走了差不多三四天！真可谓一言难尽啊！不过也算是有一点点的欣慰吧。

接下来我来分享一下我找到的图床哈。《当然我找的都是一些免费的，并且不需要备案的。》

图床成功介绍

sm.ms图床

看资料说：2015 年发布了 SM.MS 免费图床，目的就是为了解决论坛和博客分享图片的问题，并且提供一个免费的图床服务，让大家可以自由分享图片，不限制图片存储时间和不限制图片访问次数。

官网：https://sm.ms/

所以我也注册了一个账户，并且上传了一张我自己做的图片，如图：

imgurl图床

ImgURL诞生于2017年12月，是一款开源的图片管理系统（简称图床），使用PHP + SQLite3开发，您可以自行部署，也可使用https://imgurl.org/ 提供的在线服务。

官网：https://imgurl.org/

所以我也注册了一个账户，并且上传了一张我自己做的图片，如图：

清酒免费CDN图床

清酒图床貌似也是从17年开始运营至今，现在貌似使用的是腾讯COS+腾讯CDN加持，该图床加入了图片处理，内容审查等相关功能。

官网：https://pic.cdn.lkxin.cn/

所以我也注册了一个账户，并且上传了一张我自己做的图片，如图：

可用图床说明

请正确使用这些图床，毕竟这也是需要运营成本的，请善待这些宝贵资源哈。

图床失败案例

GitHub+jsDelivr+PicGo图床方案

这个办法呢以前是可行的，现在随着jsDelivr在咱们国内的备案被注销后，这个方法已经凉凉了。

Gitee+PicGo图床方案

随着gitee在2022年3月24日在其网站上加入了防盗链，gitee这个图床方案也凉凉了。不过也有大神有办法去解决，但是作为小白的我，还是算了，太折腾人了。《懂得都懂，😁》

Cloudflare Workers 反代使用 GitHub 仓库搭建的图床

这个办法我是没弄成，但是有大神已经成功了，教程地址：http://senjianlu.com/2021/12/cloudflare-workers-image/

Backblaze+vercel图床

这个就比较坑了，TM的vercel这个我已经部署成功了，但是Backblaze这个我注册不了账户，Backblaze这个注册账户应该还有东西是在咱们国内打不开的东西，直接造成Backblaze是注册不了的！但是也有大神是通过某种翻过墙头的技术，已经实现了这个图床功能。

教程地址：https://blazeb2.js.org/

以上的就是我这三四天折腾的图床方案，TM的折腾死我了，不过谁让我不想备案呢！
